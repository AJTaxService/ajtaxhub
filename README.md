# AJ TaxHub
**Burns & AJ Tax Service, L.L.C.**

Schedule C Expense Tracker — Drake Tax Ready

## Deploy Steps
1. Push this folder to GitHub
2. Import project in Vercel
3. Vercel auto-detects Vite/React — click Deploy
4. Add ajtaxhub.com in Vercel → Settings → Domains

## Local Development
```
npm install
npm run dev
```
